# BANK-LOAN-ANALYSIS-DASHBOARD

📊 Bank Loan Analysis Dashboard (Power BI)
Project Overview
This project presents a Bank Loan Report Dashboard built using Power BI to analyze loan applications, funding, repayments, and risk classification (Good vs Bad loans).
The dashboard helps stakeholders understand loan performance, customer behavior, and financial trends through interactive visuals.

🎯 Objectives
Analyze total loan applications and funding performance
Track loan repayment vs disbursement
Identify Good Loans vs Bad Loans
Monitor interest rate trends and Debt-to-Income (DTI)
Provide insights for better financial decision-making

📊 Dashboard Features
🔹 Key KPIs
Total Loan Applications: 39K
Total Funded Amount: $435.76M
Total Received Amount: $473.07M
Average Interest Rate: 12.05%
Average DTI: 13.33%

✅ Good Loans

🛠️ Tools & Technologies

Excel / CSV Dataset – Data Source

DAX – Calculated Measures & KPIs

💡 Key Insights
Majority of loans are Good Loans (~86%), indicating healthy portfolio
Fully Paid loans contribute the highest revenue
Bad Loans still account for significant financial risk
Higher DTI and Interest Rates correlate with risky loans
